# MineSweeper

## Game Guide

1. **Main Menu** (Use Keyboard)
    - Press 1 for Beginner Level
    - Press 2 for Intermediate Level
    - Press 3 for Expert Level

2. **In Game** (Use Mouse)
    - Left Click: Open Box
    - Right Click: Flag/Mark

3. **Restart** (Reset)
    - After winning or losing, press "R" on keyboard to return to main menu
